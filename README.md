# 📊 Layoffs Data Analysis (SQL)

## 📌 Problem Statement
The objective of this project is to analyze global layoffs data to uncover trends, identify the most affected companies, and understand how layoffs have changed over time.

---

## 🛠️ Skills Used
- Aggregations (SUM, MAX)
- GROUP BY
- Window Functions (DENSE_RANK)
- CTEs
- Time-based analysis (YEAR)

---

## 📈 Key Analysis Performed

### 1. Overview Metrics
- Identified maximum layoffs in a single event
- Analyzed highest layoff percentage

### 2. Company-Level Insights
- Calculated total layoffs per company
- Identified companies with highest layoffs

### 3. Time-Based Trends
- Analyzed layoffs by year
- Identified peak layoff periods

### 4. Ranking Analysis
- Ranked companies by layoffs within each year
- Identified top 5 companies contributing to layoffs annually

---

- Learned how to analyze trends using SQL
- Used window functions for ranking analysis
- Built reusable query structures using CTEs
- Understood how to extract insights from real-world datasets
